# Zipper


Zipper is a Python script that recursively zips all .txt files in a folder structure and creates an HTML file with links to the zip files.


## Prerequisites


Before you can run the Zipper program, you need to install Python3 and the required libraries. To do this, open a terminal window and run the following commands:


```bash

sudoaptupdate

sudoaptinstallpython3python3-pip

pip3installzipfileshutil

```

## Installation

To install the Zipper program, you can download the `zipper.py` file from the GitHub repository or clone the repository to your local machine. To clone the repository, open a terminal window and run the following command:

```bash

gitclonehttps://github.com/username/zipper.git

```

Replace `username` with your GitHub username.

## Usage

To use the Zipper program, open a terminal window and navigate to the directory where the `zipper.py` file is located. Then, run the following command:

```bash

python3zipper.py

```

The program will prompt you to enter the path of the source folder to process. Enter the path and press Enter. The program will then recursively zip all .txt files in the folder structure and create an HTML file with links to the zip files.

## Conclusion

Congratulations! You have successfully installed and run the Zipper program on your system. If you have any questions or issues, please feel free to contact the developer.
